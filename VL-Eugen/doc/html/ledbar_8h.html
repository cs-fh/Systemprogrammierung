void send_byte ( byte  b)

Die Funktion send_byte Die Funktion arbeitet den übergebenen Parameter bitweise ab. Dabei wird in einer Schleife jedes Mal eine '1' mit einer Bitstelle verundet. Das sorgt dafür, dass immer eine Maske auf genau die Bitstelle gelegt wird, an welcher sich die 1 auch gerade befindet, da die 1 in der Schleife immer um i Bitstellen verschoben wird.

bZahl die übertragen werden soll
